0. 클로드 코드 작업 지시

목표 — taxi_calculator_v1.2.html의 기능·로직·데이터 연동·계산식은 그대로 유지하고, 아래 디자인 스펙과 taxi_calculator_navy_yellow.html 시안의 시각 스타일(색상·타이포·컴포넌트)만 입힌다. HTML 구조/클래스명/JS는 최대한 보존하고, 스타일 레이어만 교체한다.

  • 화면 구성: ①메인 ②입력 모달(면허·차량·보험/조합·작업비) ③구성 비교 ④대출 계산기·CTA ⑤문자 신청 — 시안의 5개 화면과 1:1 대응.
  • 로고 이미지 2종을 assets/에 함께 전달: komolab_wordmark.png(컬러, 밝은 배경/헤더용), komolab_mark.png(화이트, 다크 배경/푸터용).
  • 이모지·형광색 금지. 아이콘은 흑백 라인(stroke) SVG 사용.

3. 컴포넌트 규칙

모서리 반경 (radius)

  • 화면 카드(목업 외곽): 24px · 내부 큰 카드: 14px · 항목/입력/버튼: 9~12px · 칩/뱃지: 6~8px

그림자

  • 카드: 0 4px 12px -6px rgba(22,34,74,.08)
  • CTA(앰버): 0 12px 24px -10px rgba(224,168,46,.6)
  • 선택 강조(포커스 링): 0 0 0 3px rgba(28,44,91,.1) + 1.5px Navy 보더

버튼

  • 주요 CTA(비교·저장·문자 신청): 앰버 그라디언트 배경 + Navy 텍스트(#16224A) + 앰버 그림자.
  • 선택된 옵션(연료·제조사·개월수): Navy 배경(#1C2C5B) + 흰 텍스트. 미선택: 흰 배경 + #E2E7F2 보더 + #4A5470 텍스트.
  • 보조 버튼(의견 보내기): 흰 배경 + Navy 텍스트 + #D5DCEC 보더.

항목 아이콘

  • 34×34 라운드 타일. 일반 항목: 배경 #F3F4F6 + #E5E7EB 보더 + #374151 라인 아이콘. 활성/다음 액션 항목(예: 보험/조합): 배경 #22252E + 흰 아이콘.
  • 아이콘은 24×24 viewBox stroke SVG(문서/차량/방패/공구 등). 글자 약어 사용 금지.

구성 비교표

  • 3열 그리드 1fr 64px 1fr. 중앙 세로 기둥으로 묶지 말 것 — 각 항목 행마다 독립된 Navy 라벨 칩(면허/차량/보험·조합)을 배치.
  • 좌(구성 A): 우측 정렬 + 좌측 Navy 보더. 우(구성 B): 좌측 정렬 + 우측 Amber 보더.
  • 세부 비교(상단)와 결과 요약(하단)은 결과 요약 구분선으로 영역 분리.

4. 헤더 & 푸터

  • 헤더: Navy 그라디언트 + 우상단 앰버 링 장식. 좌측에 komolab_wordmark.png + 세로 구분선 + 브랜드 스택("개인택시 양수도센터 / 한국모빌리티연구소 운영"), 우측 "?" 도움말 원. 아래 앰버 세로 룰 + 큰 타이틀 + "서울특별시 기준".
  • 총액 박스: 반투명 흰 배경. "총 예상 준비 자금"(1년 기준 표기 삭제) + 앰버 대형 숫자 + "초기 … · 월 … (12개월)".
  • 푸터: Navy 그라디언트 밴드. komolab_mark.png + 센터명 + 소개문 + 연락 칩(전화·홈페이지·카카오톡) + 구분선 + 주소 + 저작권.

5. 카피 규칙

  • '항목을 눌러 입력하세요', '1년 기준' 등의 안내 문구는 삭제.
  • 월 발생 비용에는 개월 수를 괄호로 표기 — 예: 월 483,000원 (12개월), 월 납입금액 (12개월).
  • 금액 표기: 한글 단위 혼용 허용(예: "초기 1억 1,330만"), 상세는 "113,300,000원" 원 단위.
