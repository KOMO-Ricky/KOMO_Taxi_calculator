# 개인택시 준비자금 계산기 — 디자인 핸드오프 브리핑

> 이 문서는 Claude Code 세션에서 진행한 디자인 작업의 전체 맥락 요약입니다.
> 함께 첨부하는 파일: `index.html`(현재 최신), `시안/taxi_calculator_시안_토스.html`, `시안/taxi_calculator_시안_카카오.html`

## 1. 프로젝트 개요

- **제품**: 개인택시 준비자금 계산기 (모바일 우선 단일 HTML 웹앱, max-width 480px)
- **운영사**: KOMOLAB · 개인택시 양수도센터 (한국모빌리티연구소)
- **배포**: 가비아 호스팅 예정, repo 루트 `index.html`이 배포 파일
- **저장소**: `KOMO-Ricky/KOMO_Taxi_calculator`, 작업 브랜치 `claude/repo-setup-existing-files-2n4ao5`
- **데이터**: Google Apps Script API 연동(면허시세·차량가·보험료 등), 실패 시 폴백 샘플
- **타깃 사용자**: 개인택시 면허 양수를 준비하는 고객 (중장년층 비중 높음)

## 2. 앱 구조 (레이아웃 변경 시 유지해야 할 것)

1. **진입 커버 화면** → "계산 시작하기" CTA로 본 화면 진입 (coverIn 전환)
2. **메인**: 헤더(총액) → 코치 안내 → 추천 구성 → **구성 탭 2개**(1번=초록/2번=주황, 선택 시 해당 색으로 항목 박스를 감쌈) → 항목 카드(면허/차량/보험·조합/작업비용, 클릭 시 바텀시트 모달) → 결과 요약(초기 소요비용·월 납입금액) → 구성 비교하기 버튼 → 다크 푸터
3. **구성 비교 페이지**: 좌(1번·초록)/우(2번·주황) + **가운데 세로 기둥에 항목명**, 각 항목 ✏️수정 클릭 시 모달로 직접 수정, 대출 계산기(접이식), 하단 [문자로 견적 받기]·[의견 보내기]
4. **문자 신청 페이지**: 비교표를 **캔버스로 그린 PNG 미리보기** + 이름/연락처 (솔라피 MMS 연동 예정)
5. **의견 모달**: 별점 5단계 + 추가 의견 + 성명/전화번호 + 추첨 안내

### 고정 규칙 (변경 금지)
- 1번 구성 = **초록**(#2e9e5b 계열), 2번 구성 = **주황**(#e8890a 계열) — 기능 구분색
- 폰트: **NanumSquare + Noto Sans KR**
- 비교표에서 저렴한 쪽 값 강조(▼), 조사 자동처리("~이/가")
